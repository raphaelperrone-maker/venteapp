Politique de confidentialité

Dernière mise à jour : avril 2026

1. Présentation

VenteApp est une application interne de gestion des ventes, du chiffre d'affaires journalier et du planning des collaborateurs, développée pour un usage exclusivement professionnel au sein du réseau de magasins FG Développement / Maison Perrert / Saveur Trepir.

Cette politique décrit quelles données sont collectées, comment elles sont utilisées et comment elles sont protégées.

2. Données collectées

L'application collecte uniquement les données nécessaires à son fonctionnement :

  • Données d'authentification : matricule et code PIN des collaborateurs, stockés dans Google Sheets.
  • Données de ventes : produits vendus, quantités, prix, date, nom du vendeur.
  • Données de CA journalier : montants CB, espèces, autres, magasin, employé, date.
  • Photos : tickets de caisse uploadés dans Google Drive.
  • Disponibilités planning : jours déclarés par les collaborateurs.

3. Utilisation des données

Les données collectées sont utilisées exclusivement pour :

  • Le suivi des performances commerciales par magasin
  • La gestion du planning hebdomadaire des collaborateurs
  • La consolidation des données dans le tableau de bord manager

Aucune donnée n'est vendue, partagée ou transmise à des tiers extérieurs à l'entreprise.

4. Services Google utilisés

L'application utilise les API Google suivantes, après consentement explicite de l'utilisateur :

  • Google Sheets API : lecture et écriture des données de ventes, CA, planning et collaborateurs.
  • Google Drive API : upload des photos de tickets dans un dossier dédié.

L'accès OAuth est limité aux scopes strictement nécessaires. Aucune autre donnée Google (Gmail, Agenda, etc.) n'est accessible.

5. Stockage et sécurité

Les données sont stockées dans Google Workspace (Sheets et Drive) sous le contrôle de l'administrateur du réseau. L'application elle-même ne stocke aucune donnée sur ses propres serveurs.

La configuration locale (clés de session) est stockée dans le navigateur de l'utilisateur via localStorage et peut être effacée à tout moment en se déconnectant.

6. Droits des utilisateurs

Conformément au RGPD, chaque collaborateur peut :

  • Accéder aux données le concernant en contactant l'administrateur
  • Demander la rectification ou la suppression de ses données
  • S'opposer au traitement de ses données
